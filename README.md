# 🔍 SearchIt - Professional Flutter Data Management App

SearchIt is a high-performance, professional Flutter application designed for seamless data management, real-time searching, and Excel integration. Built with **Material Design 3** and optimized for all screen sizes, SearchIt provides a premium user experience for handling structured data.

## 🚀 Key Features

### 🎬 Animated Splash Screen
- Smooth 1.8s fade-in animation.
- Professional branding with centered logo and typography.
- Orientation-aware and responsive design.

### 🏠 Feature-Rich Home Dashboard
- **Gradient Record Cards**: Immediate visual feedback on data volume.
- **Excel Integration**: One-tap upload for processing spreadsheet data.
- **Quick Guide**: Intuitive info cards for new users.
- **Feature Grid**: Easy access to core app modules.

### 🔍 Advanced Real-Time Search
- Responsive search bar with instant filtering.
- Professional data cards with rounded corners and optimized elevation.
- "No Results Found" clean UI states.

### 📊 Modern User Profile & Statistics
- Real-time data statistics and usage metrics.
- Clean, list-based feature showcase.
- Integrated pull-to-refresh for data synchronization.

### ⚙️ Dedicated Settings & Management
- Database management tools.
- Detailed app information and version tracking.
- Secure data presentation with clean typography.

## 🛠️ Technology Stack

- **Framework**: [Flutter](https://flutter.dev/) (Material Design 3)
- **Database**: `sqflite` (SQLite) for robust local storage.
- **Fonts**: `google_fonts` (Poppins) for premium typography.
- **Utility**: `excel` for spreadsheet processing.
- **Navigation**: Modern `BottomNavigationBar` using `IndexedStack` for state preservation.

## 🎨 Design System

SearchIt follows a strict professional design system:

| Element | Specification |
|---------|---------------|
| **Color Palette** | Indigo Primary (#4F46E5), Success (#10B981), Warning (#F59E0B) |
| **Typography** | Poppins (Variable weights from w400 to w700) |
| **Grid System** | 8px consistent spacing rule (8, 12, 16, 20, 24px) |
| **Border Radius** | Progressive rounding (12px, 16px, 50% for circles) |

## 📱 Device Support
Fully responsive and tested across:

## 🏁 Getting Started

1. **Prerequisites**: Ensure you have Flutter installed on your machine.
2. **Clone the Repo**:
   ```bash
   git clone https://github.com/chethucs7/searchitapp.git
   ```
3. **Install Dependencies**:
   ```bash
   flutter pub get
   ```
4. **Run the App**:
   ```bash
   flutter run
   ```
